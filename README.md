# TRACE Application

TRACE (TRack Attendance and Communicate Efficiently) is an attendance monitoring system designed to assist teachers and school administrators in tracking learner attendance and improving parent communication.

The application allows teachers to record attendance digitally while automatically informing parents of learner absences and arrival times through QR-based gate entry monitoring.

## Key Features
- Digital attendance recording by teachers  
- Automated parent notifications for absences and late arrivals  
- QR code scanning for learner gate entry  
- School administrator dashboard for attendance monitoring  
- Secure and centralized attendance database  

## Technologies Used
- JavaScript
- Firebase Firestore
- Node.js
- HTML
- CSS

The application is deployed using Render. Configuration and environment variables are managed through the Render dashboard.

This project was developed as part of a learning initiative to explore real-world school attendance systems and web deployment.
